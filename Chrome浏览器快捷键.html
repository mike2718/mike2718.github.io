Chrome浏览器快捷键

标签页和窗口快捷键

操作 快捷键
在无痕模式下打开新窗口 Ctrl + Shift + N
打开新标签页，并跳转到该标签页 Ctrl + T
重新打开之前关闭的标签页 Ctrl + Shift + T
跳转到下一个标签页 Ctrl + Tab
关闭当前标签页 Ctrl + W
关闭当前窗口 Ctrl + Shift + W

功能快捷键

操作 快捷键
打开书签管理器 Ctrl + Shift + O
打开历史记录页 Ctrl + H
打开下载内容页 Ctrl + J
打开查找栏搜索当前页 Ctrl + F

地址栏快捷键

操作 快捷键
为网址添加www和.com，然后回车 输入网址并按Ctrl + Enter
将光标移动到地址栏 Ctrl + L

网页快捷键

操作 快捷键
打印当前网页 Ctrl + P
刷新网页 F5 / Ctrl + R
刷新网页（忽略缓存） Shift + F5 / Ctrl + Shift + R
停止加载网页 Esc
显示HTML源代码 Ctrl + U
将当前网页保存为书签 Ctrl + D
全屏模式 F11
放大网页内容 Ctrl + +
缩小网页内容 Ctrl + -
将网页大小恢复为默认大小 Ctrl + 0
向下滚动屏幕 空格键
向上滚动屏幕 Shift + 空格键

鼠标快捷键

操作 快捷键
在后台标签页中打开链接 按住Ctrl点击链接
在新窗口打开并跳转到该链接 按住Ctrl和Shift点击链接
在新窗口打开并跳转到该链接 按住鼠标中键点击链接
在新窗口打开并跳转到该链接（鼠标操作） 将链接拖放到标签栏
将当前网页保存为书签 将网址拖放到书签栏中
下载链接目标 按住Alt键点击链接
查看并跳转到特定浏览记录 鼠标左键一直点击←→箭头，或者鼠标右键点击←→箭头
放大网页内容 按住Ctrl键并向前滚动鼠标滚轮
缩小网页内容 按住Ctrl键并向后滚动鼠标滚轮
