== 操作说明

[options="header"]
|===
|按键|效果

|*↑ ↓ → ←*
|进行角色移动、指令选择

|*Z、Enter、空格键*
|查看文章、调查物品、确定选择

|*X、Backspace*
|取消、打开菜单、文章快进

|*C、Shift*
|快速保存

|*F12*
|重置游戏

|*F4*
|画面尺寸变更
|===

== 攻略

游戏开始，与吉野对话。

1F职员室与彩加对话。

1F男厕所调查灭火器。

1F走廊调查大厅洗手池旁的铁柜，拿到**球棒**。
调查3F校长室的花瓶，选择破坏，拿到**紧急出口钥匙**。

2F理科室与吉野对话，调查男性尸体。

2F图书室与图书室桑对话。

3F的3-C教室与对话，触发剧情。
