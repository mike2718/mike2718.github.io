== 操作说明

[options="header"]
|===
|按键|效果

|*↑↓←→*, *2、4、6、8* （数字键盘）
|移动

|*Shift*
|快跑

|*Z*, *Enter*, *Space*
|确定、对话、调查

|*X*, *Esc*, *0* （数字键盘）
|取消、显示菜单

|*Alt+Enter*
|切换窗口/全屏模式

|*Alt+F4*
|强制退出

|*F12*
|回主菜单

|*F1*
|游戏设置
|===
